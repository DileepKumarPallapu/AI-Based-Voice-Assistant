# AI-Based Voice Assistant

## Overview
AI-Based Voice Assistant is a Python-based intelligent assistant that accepts voice commands and performs useful tasks such as answering questions, opening applications, searching the web, and responding through speech. This project demonstrates the use of speech recognition, natural language processing, and text-to-speech technologies.

## Features
- Accepts voice input from the user
- Converts speech to text
- Processes user commands intelligently
- Responds using text-to-speech
- Opens applications
- Searches the web
- Answers simple questions
- Hands-free interaction

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- pywhatkit
- Wikipedia API
- datetime
- webbrowser

## How It Works
1. The user gives a voice command.
2. The system captures audio through the microphone.
3. Speech is converted into text.
4. The command is processed.
5. The assistant performs the requested action.
6. The response is spoken back to the user.
